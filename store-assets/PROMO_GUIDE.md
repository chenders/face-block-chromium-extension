# Promotional Images Guide

Create eye-catching promotional images for the Chrome Web Store to increase downloads and visibility.

## Required Promotional Images

### Small Promo Tile (Optional)
- **Dimensions**: 440 x 280 pixels
- **Format**: PNG or JPEG
- **Purpose**: Shown in search results and category pages

### Large Promo Tile (Optional)
- **Dimensions**: 920 x 680 pixels
- **Format**: PNG or JPEG
- **Purpose**: Featured placements and detailed views

### Marquee Promo Tile (Recommended for Featured)
- **Dimensions**: 1400 x 560 pixels
- **Format**: PNG or JPEG
- **Purpose**: Top placement, homepage features, promotional banners

## Quick Start with Canva

1. Go to Canva.com
2. Create custom size (e.g., 1400 x 560 for marquee)
3. Choose a template or start blank
4. Add elements:
   - Upload extension icon
   - Add text (title, tagline, features)
   - Insert screenshot (drag from your computer)
   - Add shapes/icons for visual interest
5. Export as PNG (high quality)

## A/B Testing Ideas

Create 2-3 variations to test:

**Variation A**: Screenshot-focused
- Large screenshot showing blocking in action
- Minimal text
- Visual demonstration

**Variation B**: Feature-focused
- List of benefits
- Icons for each feature
- Text-heavy approach

**Variation C**: Privacy-focused
- Emphasis on "100% Local"
- Lock/shield imagery
- Security messaging

Test which performs better in the Chrome Web Store.
